jobSort()

- Aggregates job listings from Hacker News: Who's Hiring, Github, and Stack Overflow

Javascript, React, Redux, HTML, CSS, Node, Express, MySQL, Mocha, Chai, Enzyme

A bot checks Hacker News every hour for updates to the latest "Who's Hiring" post and stores the data in a MySQL database. The web app asynchronously calls github jobs API, scrapes Stack Overflow jobs, and queries the MySQL database for Hacker News data. The job listings are sorted by which languages the user knows and how familiar the user is with each of them. Optimized for desktop and mobile.

Yeggle

- Compares Yelp search results with Google Maps

Javascript, HTML, CSS, React, Node, Express

Calls Yelp and Google APIs to get search result data that can be filtered/sorted by various categories. The "Yeggle" button matches up Google results to Yelp results; the Yeggle algorithm utilizes coordinates, title, and address to determine if there is a match. Optimized for desktop and mobile.
