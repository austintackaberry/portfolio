jobSort()

- Aggregates job listings from Hacker News: Who's Hiring, Github, and Stack Overflow

Javascript, HTML, CSS, React, Node, Express, MySQL

A bot checks hacker news every hour for updates to Hacker News: Who's Hiring (an unformatted job posting forum), grabs the data, parses it, and stores it in a MySQL database. The web app asynchronously calls github jobs API, and scrapes Stack Overflow jobs, and queries the MySQL database for Hacker News data. Each job description is run through my regex algorithm to determine which technologies are in the description. The job listings are sorted by which languages the user knows and how familiar the user is with each of them. Optimized for desktop and mobile.

Yeggle

- Compares Yelp search results with Google Maps

Javascript, HTML, CSS, React, Node, Express

Calls Yelp and Google APIs to get search result data that can be filtered/sorted by various categories. The "Yeggle" button matches up Google results to Yelp results; the Yeggle algorithm utilizes coordinates, title, and address to determine if there is a match. User can click on map icons to get the average review rating. Optimized for desktop and mobile.
